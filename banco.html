Conceitos de Bancos de Dados

Abaixo estão os conceitos e diagramas relacionados aos bancos de dados utilizados pela nossa empresa.

O que é um Banco de Dados?

Um banco de dados é um sistema que permite armazenar, gerenciar e recuperar dados de forma eficiente.

Banco de Dados Relacional

Um banco de dados relacional organiza os dados em tabelas, onde cada tabela possui colunas e linhas, permitindo relações entre elas.

Banco de Dados Não Relacional

Os bancos de dados não relacionais, também conhecidos como NoSQL, armazenam dados em formatos flexíveis como documentos, grafos ou colunas, sem a necessidade de um esquema fixo.

O que é um Diagrama Conceitual (DER)?

Um Diagrama Entidade-Relacionamento (DER) é uma representação visual que mostra as entidades, atributos e os relacionamentos entre elas dentro de um sistema.

Diagrama DER do Exercício

O que é um Diagrama Lógico (MER)?

Um Diagrama Lógico de Entidade-Relacionamento (MER) é uma representação que detalha como as entidades se relacionam, incluindo chaves primárias e estrangeiras.

Diagrama MER do Exercício

Quais os Bancos de Dados que a Empresa Trabalha?

A empresa trabalha com diversos bancos de dados, incluindo:

  • MySQL
  • PostgreSQL
  • MongoDB
  • SQLite

O que é Dicionário de Dados?

Um dicionário de dados é um repositório que descreve a estrutura de um banco de dados, incluindo tabelas, campos, tipos de dados e relacionamentos.
